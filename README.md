# -LlamaIndex-RAG-
本项目是一个专为学术科研场景设计的高阶 RAG（检索增强生成）问答系统。系统后端基于 FastAPI 构建，深度集成了 LlamaIndex 与阿里云大模型生态（通义千问与 DashScope Embedding）。针对复杂学术文献的结构特点，项目创新性地实现了层级文档解析（基于 LlamaParse），并构建了支持动态路由与父节点自动合并的混合检索架构（FAISS 向量检索 + BM25 关键词匹配 + Rerank 重排序）。此外，系统内置了包含长期向量记忆与短期滑窗记忆的层级记忆管理机制，能够结合多轮对话上下文进行智能查询改写。它不仅能精准定位并提取论文的核心创新点、实验设计与数据分析，还配套了完整的 RAG 评估流水线，致力于为文献阅读和学术研究提供高精度的智能化辅助。
